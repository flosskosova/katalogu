# katalogu

Catalog of open source apps and tools.

A curated, editorial catalog of mainstream, trustworthy **Free and Open Source Software** (FOSS). The goal is quality and practical adoption—not exhaustive listings.

## Contents

- **[FOSS-CATALOG.md](./FOSS-CATALOG.md)** — Full catalog by category (primary reference).
- **[LICENSE-NOTES.md](./LICENSE-NOTES.md)** — How OSI-approved and recognized licenses are treated in this project.

## Principles

Entries must be genuinely open source (see license notes), actively maintained, well documented where it matters, and credible for real-world use. Source-available or non-OSI licenses that fail the project’s “genuinely open source” bar are excluded.

## Maintenance

This catalog is a living document. Verify licenses, release cadence, and project status before major adoption decisions.

## Web catalog (Next.js)

A production-oriented Next.js site lives in [`site/`](./site/): browse, filters, tool pages, comparison, SEO, and accessibility. From the `site` folder run `npm install` and `npm run dev`.

**Self-hosted VPS:** see **[DEPLOY-DOCKER.md](./DEPLOY-DOCKER.md)** for Ubuntu + Docker + Postgres + optional Caddy HTTPS.
